# Voyna 데모 시연 스크립트

> 대상: 알토대학원 벤처 스타트업 수업 / 발표 시간 5~7분  
> 마지막 업데이트: 2026-05-14

## 준비물

- 발표자 노트북 (Chrome/Edge 최신) + HDMI/USB-C 빔프로젝터 연결
- (선택) 스마트폰 — 실기기 시연 시
- 라이브 URL: **https://alexsong0674.github.io/Voyna-Demo/**
- 사전 상태: 더보기 → "데모 리셋" 누르고 시작 (Lv1, XP 0, 배지 0 초기 상태)

## 시연 동선 (약 5분 30초)

### 0:00–0:30 — 오프닝
> "여행이 게임이 된다면 어떨까요. Voyna는 GPS 기반으로 명소를 방문하면 자동으로 배지가 모이는 앱입니다. 오늘 3일 만에 만든 웹 데모를 보여드릴게요."

### 0:30–1:30 — 로그인 + 첫 화면
1. **라이브 URL 접속** → 진한 파랑 그라데이션 로그인 화면
   - 슬로건: "발걸음이 기록이 되고, 기록이 추억이 된다"
2. **"Google로 시작하기"** 클릭 → Google 페이지로 자동 이동 → 본인 계정 선택 → 자동 복귀
3. **홈 화면 진입** — "안녕하세요, [본인 이름]님" 표시
   - Lv 1 / 0 XP / 초보 탐험가
   - "아직 획득한 배지가 없어요" 안내
4. > "지금은 아무것도 없습니다. 탐험을 시작해볼까요?"

### 1:30–3:30 — 탐험 맵 + 텔레포트 (시연 핵심)
1. 하단 **🗺️ 탐험** 탭 클릭
   - 카카오맵 + 서울 31개 명소 마커
2. > "실제로는 GPS로 자동 인증되는데, 강의실이라 텔레포트 데모 모드를 켤게요"
3. 우측 하단 **✈️ 텔레포트** 버튼 클릭 → 명소 목록 패널
4. **"ASSIST 강의실"** 선택 ⭐
   - 깜짝 효과: "여기 우리가 있는 강의실도 등록되어 있어요"
   - 금색 메달 + "특별 배지 획득" 모달 + +300 XP
   - 레벨업 모달
5. 다시 텔레포트 → **"경복궁"** 선택 → +400 XP → 모달
6. 다시 텔레포트 → **"북한산"** 선택 → +400 XP → 모달
7. 다시 텔레포트 → **"N서울타워"** 선택 → +400 XP → 모달
8. 다시 텔레포트 → **"명동"** 선택 → +50 XP → 5번째 배지로 **"동네 탐험가" 칭호** 토스트 등장

### 3:30–4:30 — 배지 컬렉션
1. 하단 **🏅 배지** 탭 클릭
2. > "5개는 컬러풀하게 빛나고, 나머지 26곳은 회색 자물쇠로 잠겨 있어요. 장소 이름은 보여드려서 다음 목표가 명확하게 보이도록 했습니다."
3. 카드 구조 강조: 메달 → 장소명 (예: 경복궁) → 배지 명칭 (조선의 정궁) → 등급
4. > "자기가 가본 곳은 진짜 사진이 들어가고, 못 가본 곳은 회색조로 처리해서 '아직 미수집'이라는 게 한눈에 보입니다."

### 4:30–5:00 — 본 프로젝트 안내
1. 하단 **⚙️ 더보기** 탭
2. > "이 빌드는 데모용 웹 앱입니다. 본 프로젝트는 12주 일정으로 iOS 네이티브 앱 정식 출시를 준비하고 있고, 거기서는 시크릿 배지·구독 모델·푸시 알림 같은 풍성한 기능이 추가됩니다."

### 5:00–5:30 — 클로징
> "여행은 추억이 되지만, Voyna에서는 기록이 됩니다. 발걸음이 자산이 되는 새로운 여행 방식, Voyna. 이상입니다."

## 비상 시나리오

| 증상 | 원인 | 즉시 조치 |
|------|------|----------|
| 흰 화면 / 404 | URL 끝 슬래시 누락 | `https://alexsong0674.github.io/Voyna-Demo/` 슬래시 확인 |
| Google 로그인 안 됨 | Firebase 도메인 허용 누락 | 미리 로그인된 다른 브라우저 탭 사용 |
| 카카오맵 회색 박스 | SDK 인증 실패 | 텔레포트는 그대로 동작 (지도 없이도 배지 획득은 됨) |
| 모달이 안 뜸 | 캐시된 옛 JS | Ctrl+Shift+R 강력새로고침 |
| 네트워크 끊김 | 와이파이 문제 | localStorage 상태로 계속 동작 (지도 빼고) |

## 사전 점검 체크리스트 (발표 30분 전)

- [ ] 라이브 URL 접속 가능 — https://alexsong0674.github.io/Voyna-Demo/
- [ ] Google 로그인 동작 확인
- [ ] 카카오맵 타일 로드 성공
- [ ] 텔레포트 → ASSIST → 배지 획득 모달 정상
- [ ] 배지 탭에서 9개 사진 배지 + 자물쇠 배지 회색조 표시
- [ ] 더보기 → "데모 리셋" 누르면 초기 상태로 돌아감
- [ ] **데모 리셋 후 처음부터 한 번 더 시연** (Lv1 → 다시 시작 가능 검증)
- [ ] 빔프로젝터 미러링 화면 비율 OK
- [ ] 한글 폰트 정상 표시
- [ ] (선택) 스마트폰에서도 동일하게 동작 확인

## 데모 끝나면

- 청중에게 URL 공유 가능 (각자 폰으로 접속해서 본인 배지 모을 수 있음)
- ASSIST 강의실에 실제로 있을 때 → 실기기 GPS로 진짜 배지 획득 가능
- 본 12주 프로젝트는 [기획서 리포](https://github.com/AlexSong0674/test) 참고
